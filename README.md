# Aid_Kit

## Аптечка гомеопатических средств

Домашняя коллекция гомеопатических препаратов в формате веб-страницы.

### Описание

Простой и удобный справочник по гомеопатическим средствам для домашнего использования. Содержит информацию о препаратах от различных производителей.

### Структура проекта

```
Aid_Kit/
├── index.html          # Основная страница справочника
├── README.md           # Документация проекта
└── .gitignore          # Исключения для git
```

### Использование

Откройте файл `index.html` в любом современном браузере:

```bash
# Linux/Mac
xdg-open index.html      # Linux
open index.html          # macOS

# Или просто откройте файл через файловый менеджер
```

Все секции по умолчанию свёрнуты. Нажмите на кнопку «Развернуть» для просмотра содержимого раздела.

### Технологии

- HTML5
- CSS3
- Vanilla JavaScript
